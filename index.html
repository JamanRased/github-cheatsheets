GitHub Cheatsheets

Git Install

Tools Congfiger

  • $ git config --global user.name "[name]"
  • $ git config --global user.email "[email address]"

Repository Create & File Name Refactor

  • $ git init [project-name] - নতুন রিপোসিটোরি তৈরি
  • $ git clone [url] - ডাউনলোড করুন
  • $ git rm [file] - ওয়ার্কিং ডিরেক্টরি থেকে ফাইল মুছে এবং মুছে ফেলার পর্যায়ে
  • $ git rm --cached [file] - ফাইল মুছে ফেলা
  • $ git mv [file-original] [file-renamed] - ফাইলের নাম পরিবর্তন করা এবং কমিট

.gitignore

  • *.log
  • build/
  • temp-*
  • $ git ls-files --others --ignored --exclude-standard - ফাইলের তালিকা তৈরি

Fragments

  • $ git stash - পরিবর্তিত ট্র্যাক ফাইল স্টোর করা
  • $ git stash pop - stashed ফাইল রিস্টোর করা
  • $ git stash list - stashed changesets ফাইলের তালিকা তৈরি
  • $ git stash drop - stashed changeset ফাইল Delete
